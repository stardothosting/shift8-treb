Language files for Shift8 TREB Real Estate Listings plugin.

This directory contains translation files for the plugin.
WordPress.org will automatically generate .po and .mo files for supported languages.

For custom translations, place your .po and .mo files in this directory following WordPress naming conventions:
- shift8-treb-{locale}.po
- shift8-treb-{locale}.mo

Example:
- shift8-treb-fr_FR.po (French)
- shift8-treb-fr_FR.mo (French)
- shift8-treb-es_ES.po (Spanish)
- shift8-treb-es_ES.mo (Spanish)

The plugin text domain is: shift8-treb
